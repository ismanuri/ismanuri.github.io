Get to Know Adaptive Web Design (AWD) dan Responsive Web Design (RWB)



Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak? Adaptive dan Responsive web design adalah kemampuan suatu web untuk menyesuaikan konten yang ada didalamnya dengan media yang mengaksesnya. Sangat setuju sekali, dari sisi programmer akan memudahkan pekerjaan mereka untuk cukup membuat satu desain yang mencakup semua pc dekstop maupun mobile device.
Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan? Sudah pasti desain web sangat berubah begitu cepatnya hadirnya smartphone dan tablet. Kebutuhan masyarakat untuk dapat mengakses segala informasi kapanpun dan dimanapun membuat para desainer web harus mengakomodasi pasar besar ini. Dan perkembangan teknologi mobile mempermudah para desainer untuk cukup membuat satu website yang bisa diadaptasikan dengan bentang layar smartphone atau tablet. Salah satu contoh yang sudah menggunakan teknologi adaptive dan responsive web design ini adalah portal berita kompas.com, dimana kita tidak perlu mengetikkan m.kompas.com lagi saat kita mau mengakses portal berita tersebut.
Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut? Tentunya dengan keterbatasan fisik mereka dari bagian tubuh manapun akan sangat kesulitan bagi mereka untuk memanfaatkan dan memaksimalkan penggunaan website yang ada saat ini. Sudah pasti kita sebagai seorang designer website harus menggapai semua target pasar manapun, baik itu manusia normal maupun orang dengan phisical disabilities, dan tugas kita bagaimana membuat sebuah terobosan agar website yang kita buat akan mampu memfasilitasi mereka dengan dengan segala keterbasan yang dimiliki. Kalau untuk aplikasi mobile, kita para pengguna smartphone iOS mungkin sudah pasti mendengar apa itu Siri, yang memudahkan penggunaan smartphone Apple bagi para disable.
Ide atau konsep apa yang sesuai denganmu saat mempelajari desain web? Untuk saat ini saya belum banyak ide atau konsep untuk mempelajari desain web, mengikuti kurikulum yang disiapkan oleh hacktiv8. Seiring berjalannya waktu, semoga saya bisa menemukan konsep yang terbaik untuk saya bisa mempelajari dan menguasai web desain yang menyenangkan.
